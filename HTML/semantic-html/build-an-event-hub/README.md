This project is part of the **Semantic HTML** lessons on [freeCodeCamp](https://www.freecodecamp.org/). It demonstrates how to structure a webpage using semantic HTML5 elements to create a clean, accessible, and meaningful layout.

## Project Overview

**Event Hub** displays upcoming and past events using proper semantic tags to ensure content is organized, accessible, and easy to navigate.

## Semantic HTML Elements Used

- `<header>`: Contains the site title and navigation links.
- `<nav>`: Provides anchor links for in-page navigation.
- `<main>`: Wraps the main content of the webpage.
- `<section>`: Groups related events (Upcoming and Past).
- `<article>`: Used for individual event details.
- `<h1>`, `<h2>`, `<h3>`: Used to define a clear heading structure.
- `<p>`: Contains event descriptions and dates.
- `<img>`: Adds visuals for past events, each with an accessible `alt` attribute.

## Features

- Two sections: **Upcoming Events** and **Past Events**
- Semantic layout for better accessibility and SEO
- Simple navigation using anchor links
- Images with `alt` text for visually impaired users

## How to Use

1. Open the `index.html` file in your browser.
2. Use the navigation links to jump to different sections.
3. View event details, dates, and images.
